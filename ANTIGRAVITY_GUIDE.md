# 🛸 ANTIGRAVITY AI - Complete Operations Guide

Welcome to the **Antigravity AI Code Review Platform**. This professional-grade system integrates **Spring Boot**, **React**, and **Groq AI (Llama 3)** to provide high-velocity code diagnostics and metrics.

---

## 1. Product Overview

Antigravity AI is a split-screen intelligent IDE designed for FAANG-level code audits.

-   **Intelligent Review**: Uses the Groq API (Llama3-8b) for sub-second qualitative code analysis.
-   **Static Metrics**: Deterministic calculation of Cyclomatic Complexity, lines of code, and function density.
-   **Project Scoping**: Hierarchical organization of code snippets into searchable projects.
-   **Analytics Dashboard**: Visual health tracking using Recharts and Framer Motion for premium UX.

## 2. Step-by-Step Run Guide

### Step 1: Initialize Database
The system requires a MySQL instance. If using Docker:
```powershell
docker-compose up -d mysql
```
*Wait ~10 seconds for the database to initialize schemas.*

### Step 2: Launch Backend
Navigate to the root directory and run:
```powershell
mvnw.cmd spring-boot:run
```
-   **Endpoint**: `http://localhost:8080`
-   **Health Check**: Visit `http://localhost:8080/dashboard/summary` (Expected: JSON response with `totalProjects`).

### Step 3: Launch Frontend
Navigate to the `frontend` directory:
```powershell
cd frontend
npm install  # Only required on first run
npm run dev
```
-   **Endpoint**: `http://localhost:5173`
-   **Access**: Open your browser to `http://localhost:5173`.

## 3. API Integration & Testing

### Authentication
To get your JWT token (replace credentials with your DB user):
```bash
curl -X POST http://localhost:8080/auth/login \
     -H "Content-Type: application/json" \
     -d '{"email":"test@example.com", "password":"password"}'
```

### Dashboard Summary
```bash
curl -X GET http://localhost:8080/dashboard/summary \
     -H "Authorization: Bearer <YOUR_TOKEN>"
```

### Create Project
```bash
curl -X POST http://localhost:8080/projects \
     -H "Authorization: Bearer <YOUR_TOKEN>" \
     -H "Content-Type: application/json" \
     -d '{"name":"AI-System", "description":"Core analysis module"}'
```

### Upload & Review Snippet
1. **Upload**:
```bash
curl -X POST http://localhost:8080/code/upload \
     -H "Authorization: Bearer <YOUR_TOKEN>" \
     -H "Content-Type: application/json" \
     -d '{"fileName":"App.java", "codeContent":"void main() { for(int i=0;i<10;i++) {} }", "language":"java"}'
```
2. **Review (using the ID from step 1)**:
```bash
curl -X POST http://localhost:8080/ai/review/<FILE_ID> \
     -H "Authorization: Bearer <YOUR_TOKEN>"
```

## 4. Troubleshooting

> [!IMPORTANT]
> **CORS Issues**: Ensure the backend's `SecurityConfig.java` has `config.setAllowedOrigins(List.of("http://localhost:5173"))` configured. This is currently set correctly in your workspace.

> [!WARNING]
> **Groq API Key**: If AI analysis fails, verify that your `groq.api.key` in `src/main/resources/application.properties` is active and has credits.
